# Contributing to Data Center PUE Prediction

Terima kasih atas minat Anda untuk berkontribusi! 🎉

## 📋 Guidelines

### Melaporkan Bug

Jika Anda menemukan bug, silakan buat issue dengan:
- Deskripsi jelas tentang masalahnya
- Langkah untuk reproduce bug
- Expected behavior vs actual behavior
- Screenshots (jika ada)
- Environment details (OS, Python version, dll)

### Suggesting Enhancements

Feature requests sangat diapresiasi! Sertakan:
- Use case yang jelas
- Manfaat dari feature tersebut
- Implementasi suggestions (optional)

### Pull Requests

1. Fork repository
2. Buat branch baru (`git checkout -b feature/AmazingFeature`)
3. Commit changes (`git commit -m 'Add some AmazingFeature'`)
4. Push ke branch (`git push origin feature/AmazingFeature`)
5. Buat Pull Request

#### Code Style

- Ikuti PEP 8 untuk Python code
- Gunakan docstrings untuk functions
- Tambahkan comments untuk code yang kompleks
- Write meaningful commit messages

#### Testing

- Test code Anda sebelum submit PR
- Pastikan tidak ada breaking changes
- Update documentation jika perlu

## 🤝 Code of Conduct

- Be respectful
- Be collaborative
- Be inclusive
- Have fun!

## 📧 Questions?

Jika ada pertanyaan, jangan ragu untuk buat issue atau contact maintainer.

Thank you for contributing! ⭐
